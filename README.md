# Python Code Quiz
This is an interactive web application to help users begin to practice coding fundementals often tested in first year Computer Science courses. For now, the quiz available is to help beginners practice tracking Python variables as operations are done to them.
##Live Demo:
[![Live Demo](https://img.shields.io/badge/Live-Demo-%780000)](https://sophiaarfan.github.io/code-quiz/var-quiz.html)


## Features
- **Dynamic generation**: Questions in the quiz are randomly generated through JavaScript.
- **Variable tracking**: Challenges users to practice mental math with Python assignments.
- **Live scoring**: Displays users progress as they navigate through 10 questions.

## How to use
**Option A:**
Access via the live link: https://sophiaarfan.github.io/code-quiz/var-quiz.html
**Option B:**
1. Open `index.html` in a web browser. 
2. Click "Start Quiz" **or** navigate to the "Variable Tracking Quiz" section and start the quiz from there.
3. Type in what you think the output would be and then click the button to check your score.
4. Read the feedback given and then progress through the questions!

## Project Structure 
- `index.html`: Landing page.
- `var-quiz.html`: The quiz interface.
- `quiz.js`: The logic implemented through JavaScript for question generation.
- `css/`: Folder containing the style sheet.
